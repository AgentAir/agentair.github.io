従来版 Outlook のメール一覧表示が狭くなったときの戻し方

2026-05-13 / Outlook / Microsoft Office / Windows

Home Articles About

事象

従来版Outlookを使っているときに、メール一覧の1件あたりの表示間隔が狭くなり、 以前より見づらい状態になりました。

最初は「メール一覧の間隔が狭くなった」という問題として見ていましたが、 実際にはビューの種類、コンパクトレイアウト、メッセージプレビューの行数が組み合わさって メール一覧の高さが変わっている状態でした。

この記事の対象は、新しいOutlookではなく、従来版Outlookです。 クラシックOutlookの画面で、表示を元の見やすい状態に戻す手順を整理します。

目指した表示

目指したのは、左にフォルダー一覧、中央にメール一覧、右に閲覧ウィンドウがある 従来の3カラム表示です。

メール一覧は、1件あたりが小さすぎず、件名と本文プレビューが少し見える状態に戻したいと考えました。 最終的には、コンパクト表示を使い、メッセージプレビューを1行にする形が一番近い見た目になりました。

試したが違った設定

途中で「シングル」表示も試しました。 しかし、シングル表示にすると期待していた従来の見た目とは違う表示になり、 今回戻したかったメール一覧の雰囲気にはなりませんでした。

また、「件名だけを表示したいが、1件あたりの大きさは維持したい」という希望もありました。 ただし、メッセージプレビューをオフにすると、本文プレビューが消えるだけでなく、 メール1件あたりの行の高さも縮んでしまいました。

効果があった設定

効果があったのは、ビューをコンパクトに戻し、コンパクトレイアウトを常に使うようにしたうえで、 メッセージプレビューを1行にする設定でした。

  • 表示 → ビューの変更 → コンパクト
  • 表示 → ビューの設定 → その他の設定 → 常にコンパクト レイアウトを使用する
  • 表示 → メッセージのプレビュー → 1行
  • 表示 → 閲覧ウィンドウ → 右
  • 表示 → 間隔を狭くする → オフ

特に大きかったのは、メッセージプレビューを1行にすることです。 これにより、件名だけの詰まった表示ではなく、メール1件あたりの高さが以前に近い状態へ戻りました。

最終的な戻し手順

従来版Outlookでメール一覧の表示を戻す場合は、次の順番で確認します。

  1. Outlook上部の「表示」タブを開く
  2. 「ビューの変更」から「コンパクト」を選択する
  3. 「ビューの設定」を開き、「その他の設定」を選択する
  4. 「常にコンパクト レイアウトを使用する」を選択する
  5. 「表示」タブに戻り、「メッセージのプレビュー」を「1行」にする
  6. 「閲覧ウィンドウ」を「右」にする
  7. 「間隔を狭くする」がオンになっている場合はオフにする

この設定にすると、左にフォルダー一覧、中央にメール一覧、右に閲覧ウィンドウがある構成に戻しやすくなります。 メール一覧も、件名と本文プレビュー1行が見えるため、狭すぎない表示になります。

なぜメッセージプレビュー1行が必要だったのか

今回のポイントは、メッセージプレビューをオフにすると、メール一覧の行の高さも縮んでしまうことでした。 「本文プレビューはいらないので件名だけにしたい」と考えると、プレビューをオフにしたくなります。

しかし、今回戻したかったのは、単に本文プレビューを消した表示ではなく、 1件あたりの大きさが小さすぎない見やすい一覧表示でした。 そのため、完全に件名だけで大きさを維持するよりも、 メッセージプレビューを1行だけ表示する方が実用上の正解でした。

Outlookのメール一覧の高さは、ビュー設定だけで決まるのではなく、 コンパクトレイアウトを使うか、メッセージプレビューを何行にするかにも影響されます。

まとめ

従来版Outlookでメール一覧が狭くなった場合は、 「表示 → ビューの変更 → コンパクト」に戻し、 「常にコンパクト レイアウトを使用する」を有効にしたうえで、 「メッセージのプレビュー → 1行」にするのが効果的でした。

今回の学びは、Outlookのメール一覧の高さは、ビュー設定、コンパクトレイアウト、 メッセージプレビュー行数の組み合わせで決まるということです。 件名だけ表示にすると行の高さが縮むため、見やすさを戻す目的では 「メッセージプレビュー1行」が実用的な落としどころでした。
