テーマ：
従来版 Outlook のメール一覧表示が狭くなったときに、元の見やすい表示へ戻す手順

記事の目的：
Outlookのメール一覧で、1件あたりの表示間隔が狭くなったり、表示内容が変わってしまった場合に、従来の3カラム表示・コンパクト表示・メッセージプレビュー1行の状態へ戻す手順をまとめる。

前提：
- 対象は「従来版 Outlook」
- 新しい Outlook ではなく、クラシック Outlook の画面
- 目標は、左にフォルダー一覧、中央にメール一覧、右に閲覧ウィンドウがある表示
- メール一覧は、1件あたりが小さすぎず、件名と本文プレビュー1行が見える状態にする

今回わかったこと：
- 「シングル」表示にすると、期待した表示とは違う見た目になった
- 「コンパクト」表示が従来の見た目に近い
- 「常にコンパクト レイアウトを使用する」を有効にすると、かなり近づいた
- 最終的に「メッセージのプレビュー：1行」にすると、メール1件あたりの大きさが元に戻った
- 「間隔を狭くする」はオフにする
- 閲覧ウィンドウは右にする

最終的な設定：
1. 表示 → ビューの変更 → コンパクト
2. 表示 → ビューの設定 → その他の設定
3. 「常にコンパクト レイアウトを使用する」を選択
4. 表示 → メッセージのプレビュー → 1行
5. 表示 → 閲覧ウィンドウ → 右
6. 表示 → 間隔を狭くする → オフ

記事では、単なる手順だけではなく、以下も書いてください。
- 最初は「メール一覧の間隔が狭くなった」問題として認識していた
- 途中で「件名だけにしたいが、1件あたりの大きさは維持したい」という希望があった
- ただし、メッセージプレビューをオフにすると行の高さも縮んでしまう
- そのため、完全に件名だけで大きさを維持するよりも、「メッセージプレビュー1行」にするのが実用上の正解だった
- 最終的な理解として、Outlookのメール一覧の高さは、ビュー設定・コンパクトレイアウト・メッセージプレビュー行数の組み合わせで決まる
